# Дипломный проект по профессии «Веб-разработчик»
### Задание

Дипломный проект представляет собой создание сайта для бронирования онлайн билетов в кинотеатр и разработка информационной системы для администрирования залов, сеансов и предварительного бронирования билетов.

[Подробное описание и исходные файлы](https://github.com/netology-code/fs-2-diplom)

### Инструменты
Проект разработан на основе фреймворков Laravel и Livewire, с использованием JavaScript для всплывающих сообщений и drag-n-drop в разделе составления сетки сеансов.

* PHP 8.3.15
* Laravel 12.0
* Livewire 3.6

## Установка
Файл .env.example содержит необходимые настройки проекта, достаточно разименовать его в .env.

#### 1. Подключение зависимостей Composer
```
composer install
```
#### 2. Настройки окружения
```.env.example``` содержит необходимые настройки проекта, достаточно переименовать его в ```.env```
#### 3. Генерация ключа
```
php artisan key:generate
```
#### 4. Миграция базы данных
```
php artisan migrate
```
Файл базы не включен в репозиторий - требуется подтвердить его создание при миграции.

Интерфейс для регистрации нового пользователя не предусмотрен, создание администраторов осуществляется также при помощь миграций. Файл миграции для создания демо-пользователя: ```database\migrations\2025_07_06_004228_add_default_admin.php```
#### 5. Создание сокращенных ссылок для папок в хранилище (изображения, папки временных файлов)
```
php artisan storage:link
```
